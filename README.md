# VegaIT Timesheet

This repository contains an old full-stack **Timesheet application** project, created during Internship in VegaIT.
The solution includes a backend built with **.NET** and a frontend application, organized into multiple projects within a single solution.

## üìÅ Project Structure

The repository consists of the following main parts:

* **VegaIT.Timesheet.Api**
  ASP.NET Web API project that exposes endpoints for the Timesheet system.

* **VegaIT.Timesheet.Core**
  Core domain logic, models, and business rules.

* **VegaIT.Timesheet.Persistence**
  Data access layer responsible for database communication.

* **TimesheetFE / timesheet-app-fe**
  Frontend application for interacting with the Timesheet system.

* **TodoApi**
  Experimental or additional API project used for learning purposes.

* **WebApplication1**
  Initial or test web application project.

* **VegaIT.Timesheet.sln**
  Visual Studio solution file containing all backend projects.
